# 道路网络可视化工具

## 概述

该文件夹包含用于将成都市道路网络数据可视化为交互式地图的工具。通过使用 `folium` 库，将 GeoPackage 格式存储的道路网络数据转换为可在浏览器中查看的 HTML 交互式地图。

## 文件结构

```
.
├── readme.md          # 本说明文档
└── vis_roadnet.py     # 道路网络可视化主程序
```


## 核心功能

- **GeoPackage 数据读取**：支持读取包含道路网络数据的 `.gpkg` 文件
- **交互式地图生成**：使用 `folium` 创建可缩放、可平移的交互式地图
- **自动中心定位**：根据道路网络数据自动计算并设置地图显示中心
- **图层支持**：可指定读取 GeoPackage 中的不同图层（默认为 `edges` 层）

## 使用方法

```bash
python vis_roadnet.py
```


程序将读取指定的 GeoPackage 文件，并生成交互式 HTML 地图文件。

## 技术特点

- 使用 `geopandas` 读取空间数据
- 基于 `folium` 的 Leaflet.js 实现前端交互
- 自动计算道路网络质心作为地图中心点
- 详细的日志记录便于调试和监控

## 依赖库

- folium - 交互式地图库
- geopandas - 空间数据处理
- pandas - 数据处理
- loguru - 日志记录

## 输出结果

生成的 HTML 文件可在浏览器中打开，显示成都市道路网络的交互式地图，支持缩放、平移等操作。
